# Prerequisites:
1. `Python 3.7+`
2. `Node.js` (for frontend)
3. `Python virtualenv` (optional)

# Start Django Server (it has to start at localhost:8000):
1. `cd {project folder}`
2. `pip install -r requirements.txt`
3. `python manage.py migrate`
4. `python manage.py runserver`

# Start Vue Server
1. `cd {frontend}`
2. `npm install`
3. `npm run dev`
